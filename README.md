# Project-Hosinsa
team project used spring framework<br>

    * 회원가입/로그인/탈퇴
    * 카테고리/페이징/소팅 정렬
    * 상품검색/찜
    * 장바구니/구매(카카오페이API)
    * 상품/회원/배송정보 관련 CRUD
    * 게시판 및 댓글 CRUD 기능이 구현되었습니다.
